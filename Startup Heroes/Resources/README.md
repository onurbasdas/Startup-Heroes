# Resources

Bu klasör static resource dosyaları için kullanılır.

## Eklenebilecek Dosyalar

- **Placeholder Images**: Boş görsel durumları için placeholder görselleri
- **Localization Files**: Çoklu dil desteği için .lproj klasörleri
- **Font Files**: Özel font dosyaları (.ttf, .otf)
- **JSON Configs**: Konfigürasyon dosyaları
- **Other Assets**: Diğer static resource'lar

## Not

Şu an için bu klasör boş bırakılmıştır. İhtiyaç duyulduğunda resource dosyaları buraya eklenebilir.
